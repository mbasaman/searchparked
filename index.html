This site's mission is to seach the internet in an effort to serve the individual user
 
This is an open source project and a free service
 
All contributions are strictly restricted by the Internet Engineering Task Force's RFC 2119
 
1.001 The system SHALL NOT fail
 
2.001 The system SHALL NOT contain advertisements for any publisher
 
2.002 The system SHALL NOT engage in tracking
 
2.003 The system SHALL be non-profit
 
3.001 The system SHALL search the internet and nothing else
 
3.002 The system SHALL serve the individual user during each request and no one else
 
3.003 The system SHALL use the latest encryption technology to secure the privacy of the user
